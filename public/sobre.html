Sobre

Este projeto é um teste para estágio na empresa FATTO, visando criar uma aplicação de "Lista de Tarefas" que utilize tecnologias front-end e back-end. A aplicação armazena as tarefas em um banco de dados usando Express e Axios para as requisições no back-end, e SQLite como banco de dados. No front-end, foram utilizados JavaScript e CSS para criar uma interface amigável e responsiva, sem a necessidade de frameworks específicos. Durante o desenvolvimento, a ferramenta Postman foi empregada para validar as rotas do back-end. O projeto destaca a sinergia entre as tecnologias front-end e back-end para criar uma aplicação funcional e eficiente.
